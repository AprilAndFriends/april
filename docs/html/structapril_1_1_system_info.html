◆ localeVariant

hstr april::SystemInfo::localeVariant

Locale script and region codes as per ISO 15924 (for scripts) and ISO 3166 (for regions).

Note
If available, the "script" (ISO 15924) will always be first and the region (ISO 3166) will always be second. In that case they are separated by a "-" (dash) character.
