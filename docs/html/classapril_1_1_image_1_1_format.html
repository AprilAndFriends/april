Detailed Description

Gets the BPP.

Defines the pixel format of image data.

Returns
The BPP. Gets the index of the red channel.
The index of the red channel. Gets the index of the green channel.
The index of the green channel. Gets the index of the blue channel.
The index of the blue channel. Gets the index of the alpha channel.
The index of the alpha channel. Gets all channel's indices.
Parameters
[out]redIndex of red channel.
[out]greenIndex of green channel.
[out]blueIndex of blue channel.
[out]alphaIndex of alpha channel.
Note
Some formats are intended to improve speed with the underlying engine if really needed. X formats are always 4 BPP even if that byte is not used.
