enum april::MessageBoxStyle

Defines possible display styles of a message box.

Note
On some OSes this affects what icon is displayed within the message box.
Enumerator
MESSAGE_STYLE_NORMAL 

The standard display.

MESSAGE_STYLE_INFO 

Special informational display.

MESSAGE_STYLE_WARNING 

Display as warning.

MESSAGE_STYLE_CRITICAL 

Display as critical.

MESSAGE_STYLE_QUESTION 

Display as question.

MESSAGE_STYLE_MODAL 

Force modal message box.

Note
Modal means that the user cannot interact with the underlying window until the message box is closed.
Not available on all OSes and some OSes have always-modal message boxes.
MESSAGE_STYLE_TERMINATE_ON_DISPLAY 

Close underlying application before showing the message box.

Note
This might not work the same way in all OSes.

Displays an OS message box.

Parameters
[in]titleText displayed in the title bar.
[in]textText displayed within the message box.
[in]buttonMaskButtons which should be displayed.
[in]styleStyle of the message box.
[in]customButtonTitlesCustom texts for the buttons.
[in]callbackCallback function to call once the message box was dismissed. Can be NULL if not needed.
Note
"customButtonTitles" may not work on all OSes.
Depending on the OS, this function may return immediately. Write your code to assume that this function is asynchronous to avoid problems.
Expect that "callback" will be called asynchrously once the message box is dismissed.
