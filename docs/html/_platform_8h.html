Displays an OS message box.

Parameters
[in]titleText displayed in the title bar.
[in]textText displayed within the message box.
[in]buttonsButtons which should be displayed.
[in]styleStyle of the message box.
[in]customButtonTitlesCustom texts for the buttons.
[in]callbackCallback function to call once the message box was dismissed. Can be NULL if not needed.
[in]modalForce modal message box.
[in]terminateOnDisplayClose underlying application before showing the message box.
Note
Modal means that the user cannot interact with the underlying window until the message box is closed. It's not available on all OSes and some OSes have always-modal message boxes.
terminateOnDisplay may not work the same way in all OSes.
"customButtonTitles" may not work on all OSes.
Depending on the OS, this function may return immediately. Write your code to assume that this function is asynchronous to avoid problems.
Expect that "callback" will be called asynchrously once the message box is dismissed.
