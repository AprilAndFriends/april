Member Data Documentation

static const LoadMode april::Texture::LoadMode::Async
static

Loads the texture asynchronously right away and uploads the data to the GPU as soon as it is available before the next frame.

Note
If the texture is used before it loaded asynchronously, it cannot be uploaded to the GPU and will not be rendered properly.
static const LoadMode april::Texture::LoadMode::AsyncDeferredUpload
static

Loads the texture asynchronously right away, but it will upload the data to the GPU on the first use.

Note
If the texture is used before it loaded asynchronously, it cannot be uploaded to the GPU and will not be rendered properly.
